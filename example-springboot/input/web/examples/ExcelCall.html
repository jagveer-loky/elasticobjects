Excel Calls

With the ==>{ConfigLinkCall->ModelConfig, XlsxReadCall}. one can read Excel files.

In the following examples the file ==>{ConfigLinkCall->FileConfig, AnObject.xlsx:test}. is used.

The result in a table
==>{XlsxReadCall->AnObject.xlsx:test, xyz}. ==>{TemplateResourceCall->table.tpl, xyz}. The table that you see beforehand is created by the following template sequence with ==>{ConfigLinkCall->FileConfig, table.tpl}. within the ==>{GithubLinkCall->NONE, SPI|web.examples|ExcelCall.html}. content: In the last example you can see the post version of these calls. ==>{ConfigLinkCall->ModelConfig, AnObject}. is a test class.
Simple Call

In the following example will read the file as a list of maps.

Since the xlsx configuration defines "rowHead" with 0, the value entries will be mapped as a list of map entries using the first row as a key.

Setting Parameters

This example will read the file as a list of lists with "rowHead" set to -1.

The header row entry in the configuration will ignored, the first value entry together with the header row will mapped as list of list.
