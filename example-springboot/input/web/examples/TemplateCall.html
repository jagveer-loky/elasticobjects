Csv, Excel and DB Together

All implemented list read calls are integrated in that editable template:

  • #{ConfigLinkCall->ModelConfig, CsvSimpleReadCall}. set #{ConfigLinkCall->FileConfig, AnObject.csv}. to "data/csv"
  • #{ConfigLinkCall->ModelConfig, XlsxReadCall}. with #{ConfigLinkCall->FileConfig, AnObject.xlsx:test}. to "data/xlsx"
  • #{ConfigLinkCall->ModelConfig, DbSqlReadCall}. with #{GithubLinkCall->NONE, EODBR||DbSqlConfig.json}. to "data/db"

#{ConfigLinkCall->ModelConfig, TemplateResourceCall}. will the render the data with #{ConfigLinkCall->FileConfig, table.tpl}.
