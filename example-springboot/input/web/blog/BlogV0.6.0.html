New Markups

The task is to integrate typed JSON in some arbitrary text with a character combination for start and stop which has low probability to clash with existing programming code. Since it's content core is typed JSON I put curly brackets around.

The end pattern is a dot outside the bracket. If the markup has a content, it will set by a pipe.

To emphasize the directives type I put a @ before. Every space and tab before this pattern and after a new line will be ignored. This makes the template more readable.
